# 菜谱

既想写给2020的你，也想贪心写给10年、20年后的工程师们，现在的你站在浪潮之巅，面对魔幻的互联网世界，很容易把一条河流看成整片大海。未来的读者已经知道了这段技术发展历史，但难免会忽略一些细节。如果未来的工程师们真的创造出了时间旅行机器，可以让你回到2020年。那么这个项目就是你和当年工程师们的接口暗号，你能感知到他们在这个时代的键盘上留下的余温。

(上段文字灵感来源于何帆《变量》)
